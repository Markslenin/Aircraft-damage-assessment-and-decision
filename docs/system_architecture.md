# System Architecture                                                                                                              
                                                                                                                                  
                                                                                                                                  ```mermaid                                                                                                                         
                                                                                                                                  flowchart LR                                                                                                                       
                                                                                                                                    A[Damage Parameterization] --> B[Nominal Predictor]                                                                              
                                                                                                                                    B --> C[Residual Generator / Filter]                                                                                             
                                                                                                                                    C --> D[Identifier]                                                                                                              
                                                                                                                                    D --> E[Controllability Evaluation]                                                                                              
                                                                                                                                    E --> F[Trim Feasibility]                                                                                                        
                                                                                                                                    F --> G[Decision Manager]                                                                                                        
                                                                                                                                    D --> H[Visualization Interface]                                                                                                 
                                                                                                                                    G --> H                                                                                                                          
                                                                                                                                    G --> I[Closed-loop Evaluation]                                                                                                  
                                                                                                                                    D --> I                                                                                                                          
                                                                                                                                  ```                                                                                                                                
                                                                                                                                  
                                                                                                                                  The online chain is separated from offline training so the deployed model path can evolve without changing the assessment backbone.
